Dona nenê - Restaurante

Nesta publicação, mergulhamos em um emocionante projeto de aprendizado, onde criamos um website completo para um restaurante fictício chamado "Restaurante Estudo". Acompanhe-nos nessa jornada enquanto exploramos a construção do site, passo a passo, utilizando tecnologias web fundamentais.

  • Nota:
  • Cliente: Projeto universitário
  • Completed: dez 2022

Estrutura básica do site com HTML e aplicando estilos com CSS.br
CherryPy e Python: Utilizando o framework CherryPy em Python para configurar um servidor web e gerenciar as páginas.
SQLite para armazenar informações de reservas.

Presidente Prudente - LOnga Exposição

Nesta deslumbrante foto capturada com um drone durante a noite, somos transportados para uma visão aérea deslumbrante de uma cidade moderna. O cenário noturno revela uma metrópole iluminada e pulsante, com arranha-céus que se erguem majestosamente até o céu.

  • Nota:
  • Cliente: Projeto Pessoal
  • Completed: Ago 2023

A longa exposição da imagem cria um efeito mágico: os rastros de luzes dos carros que se movem pelas ruas da cidade parecem fitas de energia, pintando um quadro vibrante e dinâmico. Cada rastro de carro é como um traço de vida em movimento, adicionando um elemento de movimento e energia à cena.
